## 📋 About

A modern, fully internationalized (🇫🇷 / 🇬🇧) personal portfolio built with **Vite + TailwindCSS**. It showcases my expertise in software engineering, blockchain development, and business architecture.

### Key Features

- 🌍 **Auto i18n** — Detects browser language (FR/EN) automatically with a manual toggle and `localStorage` persistence
- 🐙 **Live GitHub Projects** — Dynamically fetches all public repositories from the GitHub API
- 📖 **README Viewer** — In-site Markdown renderer for projects without a description
- 📥 **CV Download** — One-click PDF download of my official resume
- ✨ **Premium UI** — Manrope typography, clean white design, micro-animations, fully responsive
